IDEA integration

Just install the Lilith plugin in IDEA and you'll be able to jump to the source of an event using a SHIFT-doubleclick or by clicking on a stacktrace element in either the callstack or a Throwable.

To install the plugin simply open Settings in IDEA (Cmd+Alt+S), select Plugins in IDE Settings, select the Available tab and select Lilith. Then you have to click on "Update plugin".
