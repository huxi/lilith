Customizing details view of event.

As of version 0.9.27 of Lilith, it is possible to completely customize the details view of an event. The xhtml is generated by a Groovy script contained in "[lilith.home]/detailsView/detailsView.groovy".

By default, [lilith.home] equals [user.home]/.lilith but this can be changed in preferences.

If you destroy this file, somehow, then simply delete it entirely and restart Lilith. The original detailsView.groovy file will be restored.

See http://groovy-lang.org/ for help on Groovy.

You can reinitialize the details view in the Troubleshooting preferences.

Additionally, certain details view options are available in General preferences.
