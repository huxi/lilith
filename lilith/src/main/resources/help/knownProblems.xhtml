Known Problems

  • The 'Send to' functionality isn't working as expected if the network infrastructure changes. It should work but doesn't. I think it's a JmDNS problem but I have to investigate further.
  • The 'Edit groovy!' functionality is currently broken. Just don't use it. See http://jira.codehaus.org/browse/GROOVY-2790 for more information.
  • It's possible to save the current filter using *command* I but you can't use them at the moment ;)
  • Copy to clipboard does not work in details view. This is a limitation of Flying Saucer that's used as XHTML renderer. Instead, you can copy parts or all of the event using the context menu.
  • 'Copy event' was meant to copy the event as HTML but this is severely broken in Java. There's no workaround available but this will *probably* be fixed in JDK7. So for now, 'Copy event' copies the XHTML code as text...
  • 'Clean all inactive logs' is currently executed on the EventDispatchThread. This means that it might look like the UI is frozen if there are lots of files. Just be patient for now, this will be changed in the future.
