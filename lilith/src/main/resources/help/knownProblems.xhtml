Known Problems

  • Preferences dialog is obviously not finished right now :)
  • The 'Send to' functionality isn't working as expected if the network infrastructure changes. It should work but doesn't. I think it's a JmDNS problem but I have to investigate further.
  • The 'Edit groovy!' functionality is currently broken. Just don't use it. See GROOVY-2790 for more information.
  • Copy to clipboard does not work in details view. This is a limitation of Flying Saucer that's used as XHTML renderer. This problem seems to be fixed in the next version of Flying Saucer but it hasn't been released yet.
    See bug #172.
    In the meantime, you can copy parts or all of the event using the context menu.
  • 'Copy event' was meant to copy the event as HTML but this is severely broken in Java.
    There's no workaround available but this will *probably* be fixed in JDK7. So for now, 'Copy event' copies the XHTML code as text...
    See bug #6392086.
