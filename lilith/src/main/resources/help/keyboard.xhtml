ToC

Keyboard

^ represents the system dependent command key, e.g. "Ctrl" on Windows and "cmd ⌘" on Mac

⇧ represents "Shift".

↵ represents "Enter" or "Return".

General

F1
Show help...
^⇧ O
Open inactive log...
^⇧ X
Remove all inactive logs
^ Q
Exit. Quit. No, I won't ask you for permission. I will ask for permission if you make me do so, though. :p

View related

^ 1
Opens the global logging view.
^ 2
Opens the global access view.
^ 0
Opens the internal Lilith logging view.
^ R
Remove inactive views.
^ T
Toggles tailing ("scroll to bottom"-funtionality).
^ P
Pause/Unpause view.
If the view is paused, events are still received in the background, i.e. no events are lost while the view is paused.
^⇧ A
Attach/detach view.
^⇧ D
Disconnect - terminates the connection.
^ M
Focus message.
^ E
Focus events.
^ B
Edit source name of the current view.
^ I
Add the current filter.
^ . (also ^ [mouse wheel up])
Zoom in
^⇧ . (also ^ [mouse wheel down])
Zoom out

Tab related

^ W
Close the current tab.
The unfiltered tab is not closeable.
^⇧ W
Close all but the current tab.
The unfiltered tab is not closeable.
^⇧Alt W
Close all tabs.
The unfiltered tab is not closeable.
^ J
Select previous tab.
^ K
Select next tab.

Find related

^ F
Show find panel and focus find textfield.
^⇧ F
Reset find.
^ G
Find previous match.
^⇧ G
Find next match.
^ L
Find previous match of any active condition.
^⇧ L
Find next match of any active condition.
Esc (if find panel is visible)
Close find panel.
↵ (if in find textfield)
Create a new tab with the current filter.
⇧ ↵ (if in find textfield)
Replace filter of the current tab with the current filter.
Opens a new tab, too, if the current tab is the unfiltered tab.
