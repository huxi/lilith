The story of Lilith

I finished a quick LogBACK compatibility hack for Chainsaw on 2007-05-19 and realized that it wouldn't do at all. It exploded under heavy load and would freeze our webapp in the process. I could fix this specific problem with another quick hack as an interim solution but Chainsaw would've required a *significant* amount of cleanup work.

So I decided to just do a fresh, clean and modern (JDK 1.5) implementation of a LogBACK Event Viewer instead.

The first functional but very limited prototype was released on 2007-06-25 and is under heavy production use ever since that point of time.

Chainsaw

Chainsaw v2 is a companion application to Log4j written by members of the Log4j development community. Like a number of Open Source projects, this new version was built upon inspirations, ideas and creations of others. Chainsaw v2 has it's roots from the original Chainsaw utility written by Oliver Burn, and with inspiration from the Log Factor 5 utility contributed by ThoughtWorks Inc.

LogFactor5

This is the predecessor of Chainsaw. I wasn't able to find any current documentation about it but it is still contained in the log4j release.

To start it, execute java -cp log4j-1.2.15.jar org.apache.log4j.lf5.StartLogFactor5

Vigilog

This application is a viewer for Log4j and java.util.logging XML log files.

Java Log Viewer (JLogViewer)

JLogViewer is a lightweight pure java graphical application to easily view and manage log files generated by the "java.util.logging" package. So far it can handle only file formatted with the "XMLFormatter" class.

Otros log viewer/parser

Useful software for analysing applications logs/traces. Includes parsers for java.util.logging and log4j layouts.
